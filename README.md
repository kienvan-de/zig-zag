# zig-zag

A lightweight LLM proxy that implements OpenAI-compatible API endpoints and translates requests to dedicated LLM model APIs or orchestration APIs.

## Overview

zig-zag acts as a translation layer between OpenAI-compatible clients and various LLM providers, allowing seamless integration with different model APIs through declarative mapping configurations.

## Features

- **OpenAI-Compatible API**: Drop-in replacement for OpenAI API clients
- **Multi-Provider Support**: Translate requests to different LLM model APIs
- **Declarative Configuration**: Extend functionality through mapping configs
- **Orchestration Support**: Compatible with LLM orchestration APIs
